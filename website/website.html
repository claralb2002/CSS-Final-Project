DATA

Constructing Dataset

We began by identifying sources for song metadata, charts, and lyrics.

Fetch Songs

Using APIs, we collected top charting songs across decades, capturing attributes like artist, genre, and release year.

Fetch Lyrics

We fetched lyrics from online sources and ensured each track had valid and complete text for further analysis.

Data Cleaning and Preprocessing

Preprocessing included normalization, punctuation removal, and handling missing data.

Lyrics Tokenizing

Lyrics were tokenized using standard NLP pipelines including stopword filtering and lemmatization.

Dataset Stats

Our dataset consists of 5,000 songs across 30 years with full metadata and cleaned lyrics.
